# Weather App

## Overview
The Weather App is a comprehensive tool designed to provide weather information and forecasts for various locations worldwide. It utilizes data from [OpenWeatherMap](https://openweathermap.org/) and aims to deliver a user-friendly experience for viewing and understanding weather conditions.

## Features
- **Real-time Weather Data**: Get the latest weather conditions for your chosen location.
- **5-Day Forecast**: View weather forecasts for the next five days, including temperature, humidity, and precipitation.
- **Search Functionality**: Easily search for weather data in various cities.
- **User-Friendly Interface**: Intuitive design for easy navigation and information access.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **API**: OpenWeatherMap API

## Getting Started
1. **Clone the repository**:
   ```bash
   git clone https://github.com/wolfSkullCave/Weather.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd Weather
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the application**:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000` to access the app.

## Usage
- Enter a city name into the search bar and click 'Search' to view the current weather and forecast.
- Use the provided buttons to switch between the current weather and the 5-day forecast.
